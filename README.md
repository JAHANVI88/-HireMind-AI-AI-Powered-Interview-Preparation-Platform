## 📖 Overview

HireMind AI is a next-generation interview preparation platform that leverages Generative AI and Retrieval-Augmented Generation (RAG) to deliver personalized interview experiences.

The platform analyzes uploaded resumes, extracts relevant skills and experiences, generates customized interview questions, conducts mock interviews, and provides AI-driven feedback to help candidates improve their performance.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/JAHANVI88/hiremind-ai.git
cd hiremind-ai
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
MONGODB_URI=
GEMINI_API_KEY=
PINECONE_API_KEY=
JWT_SECRET=
RAZORPAY_KEY_ID=
RAZORPAY_KEY_SECRET=
FIREBASE_CONFIG=
```

### Run Development Server

```bash
npm run dev
```
